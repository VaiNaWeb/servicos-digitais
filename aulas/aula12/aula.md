# Criando um banco de dados

Um banco de dados é uma coleção de dados relacionados entre si, como uma grande caixa em que armazenamos e organizamos todas as nossas informações de forma muito simples e inteligente. Cada banco de dados tem suas características e conceitos, mas geralmente todos são criados com o propósito de armazenar e organizar dados para que possamos acessar nossas informações de forma eficiente.
